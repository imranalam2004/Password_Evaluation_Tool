# 🛡️ Sentinel: Advanced Password Intelligence Tool

Sentinel is a high-performance, desktop-grade password evaluation tool designed with a focus on modern aesthetics and cryptographic precision. It provides real-time feedback on password complexity, visualizes strength through a dynamic clock-gauge, and calculates exact "Time to Crack" estimations using industrial-standard brute-force algorithms.

## ✨ Key Features

- **💎 Glassmorphism UI**: A stunning, semi-transparent interface that feels alive with micro-animations and responsive layouts.
- **🕒 Dynamic Clock-Gauge**: Real-time visual feedback on your password's strength, moving with every keystroke.
- **⚡ High-Precision Analysis**: Beyond simple length checks—Sentinel analyzes entropy, character diversity, and common patterns.
- **🔬 Precise Crack Estimation**: Calculates how long it would take an attacker to crack your password, from milliseconds to trillions of years.
- **💡 Smart Suggestions**: Provides actionable advice to strengthen weak passwords on the fly.

## 🚀 Getting Started

### Prerequisites

- **Python 3.x**: Required for the backend calculation engine.
- A modern web browser (Chrome, Edge, or Firefox recommended).

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yadavnikhil03/Password_evaluation_tool.git
   cd Password_evaluation_tool
   ```

2. **Run the Local Server**
   Sentinel uses a Python-based backend to ensure secure and fast processing.
   ```bash
   python main.py
   ```

3. **Open the App**
   Navigate to `http://localhost:8000` in your browser.

## 🛠️ Technology Stack

- **Frontend**: Vanilla HTML5, CSS3 with Glassmorphism, JavaScript ES6+.
- **Backend**: Python 3 (Fast, lightweight server and calculation engine).
- **Styling**: Pure CSS for maximum performance and flexible animations.
